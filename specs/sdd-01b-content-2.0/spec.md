# SDD-01b — 랜딩 페이지 콘텐츠 2.0 기획 반영

## 미션
SDD-01에서 UI Kit으로 이식한 랜딩 페이지의 텍스트 콘텐츠를 MIND BREEZE 2.0 종합기획서(`docs/MIND_BREEZE_2.0_종합_기획.md`)에 맞게 수정한다. **디자인/레이아웃/컴포넌트 구조는 그대로 유지하고 텍스트만 교체.**

## MIND BREEZE 2.0 핵심 포지셔닝
- **뇌파 기반 심리상담·명상 통합 서비스 플랫폼**
- "LINK BAND가 뇌파를 측정하고, AI가 분석하여 심리상담사와 명상 지도사에게 과학적 인사이트를 제공합니다"
- 서비스 모델: B2B (상담사/명상 지도사가 고객, 내담자는 최종 사용자)
- LINK BAND는 선택적 사용 (opt-in) — 미착용 시에도 AI 기록·요약 등 핵심 기능 제공
- AI 기반 자동 녹음·STT·요약·리포트
- 대상: 임상 심리상담사, 최면 심리상담사, 명상 지도사
