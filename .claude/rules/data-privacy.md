# 데이터 프라이버시 + 보안 규칙

## 민감 데이터

- **뇌파(EEG) 데이터**: 개인 건강 정보. 저장 시 암호화, 접근 로그 필수.
- **상담 내용**: 음성 녹음 + STT 텍스트. 상담사-내담자 간 비밀 유지.
- **개인 식별 정보**: 이메일, 이름, 생년월일. 최소 권한 접근.

## 동의(Consent) 플로우

1. **LINK BAND 착용 동의**: 세션 시작 전 명시적 opt-in. 미동의 시에도 세션 진행 가능.
2. **음성 녹음 동의**: 세션 시작 전 별도 동의. 거부 시 수동 기록 모드.
3. **데이터 보관 동의**: 보관 기간, 목적, 삭제 권한 고지.

## 기술적 보호

- **전송**: TLS 1.3 필수. WebSocket도 wss://.
- **저장**: S3 서버사이드 암호화(AES-256). DB 컬럼 레벨 암호화(민감 필드).
- **접근 제어**: JWT + 역할 기반. EEG 데이터는 본인 세션 + 담당 상담사만 접근.
- **감사**: 모든 데이터 접근 로깅. 6개월 보관.

## 법적 준거

- **대한민국**: 개인정보보호법, 생체정보 보호 가이드라인.
- **EU (향후)**: GDPR. 데이터 주체 권리(열람, 정정, 삭제, 이전).
- **의료기기**: LINK BAND는 의료기기 아님. "진단/치료 목적 아님" 고지 필수.
