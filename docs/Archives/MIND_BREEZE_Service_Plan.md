## 1. 프로젝트 개요

> **프로젝트 비전**
>
> 심리상담사와 명상 지도사의 업무 효율을 AI 기반 자동 기록과 요약으로 혁신하고, 선택적으로 룩시드랩스의 LINK BAND 뇌파 측정을 결합하여 내담자의 상태를 과학적으로 파악할 수 있는 통합 멘탈 케어 플랫폼. LINK BAND 없이도 상담 기록, AI 요약, 리포트 등 핵심 기능을 온전히 이용할 수 있으며, LINK BAND 착용 시 실시간 뇌파 모니터링과 개인화 피드백이 추가로 제공됩니다.

### 1.1 배경

기존 MIND BREEZE 서비스는 뇌과학 기반 명상 클래스를 중심으로 운영되고 있으며, LINK BAND를 통해 명상 중 뇌파를 측정하고 AI가 실시간 분석 및 피드백을 제공하는 서비스입니다. 본 프로젝트는 이를 확장하여 임상 심리상담, 최면 심리상담, 명상 수업을 통합 관리할 수 있는 웹 기반 플랫폼을 구축합니다.

### 1.2 서비스 범위

| 구분 | 내용 |
|------|------|
| 서비스 유형 | 임상 심리상담 (1:1) / 최면 심리상담 (1:1) / 명상 수업 (다수) |
| 플랫폼 | 웹 기반 (Web Bluetooth API 활용) |
| 하드웨어 | LINK BAND 2.0 (2채널 EEG, BLE 연동) — 선택적 사용 |
| 주요 사용자 | 상담사 / 명상 지도사 / 내담자 / 수업 참여자 |
| AI 기능 | 상담 기록 자동 요약 / 실시간 뇌파 분석 / 개인화 리포트 생성 |

## 2. 핵심 문제 정의 및 솔루션

현재 상담사와 명상 지도사가 경험하는 핵심 문제점과 이에 대한 MIND BREEZE의 솔루션을 정리합니다.

| # | 문제점 | MIND BREEZE 솔루션 | 핵심 기능 |
|---|--------|-------------------|----------|
| 1 | **수업/상담 관리 불편** — 수업 생성, 일정 조율, 내담자 알림 등 수작업 과다 | 통합 예약/일정 관리 시스템으로 수업 생성부터 알림까지 원스톱으로 처리 | 수업/상담 대시보드, 예약 관리, 내담자 관리 |
| 2 | **기록 작성의 번거로움** — 상담 내용, 명상 방식 등 정확한 기록이 남지 않음 | 음성 녹음 기반 자동 기록 + AI 요약으로 상담사 부담 최소화 | STT 변환, AI 요약, 상담/수업 기록지 자동 생성 |
| 3 | **실시간 상태 파악 불가** — 명상/상담 중 내담자 변화를 객관적으로 파악하기 어려움 | LINK BAND 연동(선택)으로 상담사가 내담자의 뇌파 상태를 실시간 모니터링 | 실시간 EEG 대시보드, 집중/이완/스트레스 지수 시각화 |
| 4 | **리포트 부재** — 세션 종료 후 참여자에게 제공할 결과물 없음 | AI가 세션 데이터 기반 개인화 리포트 자동 생성 및 발송 (LINK BAND 착용 시 뇌파 분석 포함) | AI 요약 리포트(기본) + 뇌파 분석 리포트(선택) + 추이 분석 |
| 5 | **지속 관리 어려움** — 상담 간 기간에 내담자 자가 관리 방법 부재 | 상담사가 처방한 셀프 트레이닝 + LINK BAND 데이터 연동(선택) | 셀프 트레이닝 모드, 일일 기록, 다음 상담 연계 |

## 3. 사용자 유형 정의 (Persona)

### 3.1 상담사 / 명상 지도사 (전문가)

**역할:** 세션을 생성하고, 내담자를 관리하며, 실시간으로 모니터링하고, 기록을 관리하는 주체

- **임상 심리상담사:** 1:1 상담 세션 운영, 내담자의 심리 상태 추적 및 기록
- **최면 심리상담사:** 1:1 최면 세션 운영, 내담자의 의식/무의식 상태 모니터링 권장
- **명상 지도사:** 다수 참여자 수업 운영, 전체 참여자의 실시간 상태 조회 권장

### 3.2 내담자 / 참여자 (클라이언트)

**역할:** 세션에 참여하고, 동의 시 LINK BAND를 착용하여 뇌파 데이터를 제공하며, 리포트를 수신하고, 셀프 트레이닝을 수행하는 주체 (LINK BAND 미착용 시에도 상담/수업 참여 및 AI 요약 리포트 이용 가능)

- **상담 내담자:** 예약된 세션에 참여, 개인 리포트 확인, 셀프 트레이닝 수행
- **명상 참여자:** 수업에 참여, 그룹 리포트 확인, 셀프 명상 트레이닝 수행

## 4. 핵심 기능 명세

### 4.1 수업/상담 관리

상담사와 명상 지도사가 세션을 생성하고 내담자를 관리하는 핵심 기능입니다.

#### 4.1.1 세션 생성 및 관리

| 항목 | 설명 |
|------|------|
| 세션 유형 선택 | 임상 심리상담 / 최면 심리상담 / 명상 수업 중 선택 |
| 일정 설정 | 날짜, 시간, 소요 시간, 반복 설정 (주간/월간) |
| 참여자 초대 | 내담자 검색 및 초대 (1:1 상담) / 수업 링크 공유 (명상 수업) |
| 알림 발송 | 예약 확정 시 자동 알림 (Email/웹 Push), 세션 전 리마인더 |
| 세션 상태 관리 | 예정 / 진행중 / 완료 / 취소 상태 관리 |

#### 4.1.2 내담자/참여자 관리

- **내담자 프로필:** 기본 정보, 상담 이력, 뇌파 데이터 추이(LINK BAND 착용 시), 특이사항 메모
- **상담 이력 타임라인:** 전체 세션 이력을 시간순으로 확인
- **태그 및 그룹 관리:** 내담자를 태그로 분류하여 관리

### 4.2 자동 기록 및 AI 요약

세션 중 발생하는 모든 대화와 활동을 자동으로 기록하고 AI가 요약하는 기능입니다.

#### 4.2.1 음성 기반 자동 기록

- 세션 시작 시 음성 녹음 자동 시작 (동의 후)
- STT (Speech-to-Text) 변환으로 전체 대화 텍스트화
- 화자 분리 (Speaker Diarization): 상담사/내담자 발화 구분
- 타임스탬프 기반 뇌파 데이터와 동기화 (LINK BAND 착용 시)

#### 4.2.2 AI 요약 및 기록지 생성

| 세션 유형 | AI 요약 항목 | 기록지 포함 내용 |
|----------|-------------|----------------|
| 임상 심리상담 | • 주요 호소 사항 요약 | • 세션 요약 / 뇌파 분석 결과(선택) |
| | • 내담자 감정 변화 키워드 | • 다음 세션 제안사항 |
| | • 상담사 개입 포인트 | • 상담사 메모 영역 |
| 최면 심리상담 | • 최면 단계별 뇌파 변화(선택) | • 최면 유도 과정 기록 |
| | • 의식/무의식 전환 구간(선택) | • 뇌파 변화 그래프(선택) |
| | • 유도 기법 및 반응 기록 | • 다음 세션 권고사항 |
| 명상 수업 | • 수업 진행 흐름 요약 | • 수업 콘텐츠 및 진행 방식 |
| | • 참여자 평균 이완도 추이(선택) | • 참여자별 반응 데이터(선택) |
| | • 수업 구간별 효과 분석 | • 수업 효과 통계 |

### 4.3 실시간 뇌파 모니터링 (선택 기능)

LINK BAND를 통해 측정된 뇌파 데이터를 실시간으로 시각화하여 상담사/명상 지도사에게 제공하는 기능입니다. LINK BAND 착용은 필수가 아닌 선택 사항으로, 내담자/참여자의 동의를 받은 경우에만 활성화됩니다.

> **LINK BAND 착용 여부에 따른 서비스 범위**
>
> **▶ LINK BAND 착용 시 (Opt-in)**
> - 실시간 뇌파 모니터링 대시보드 (집중도/이완도/스트레스 지수)
> - 뇌파 분석 기반 리포트 (세션 전후 변화, 추이 분석)
> - 상태 변화 알림 및 시점 태그 기능
> - 셀프 트레이닝 시 뇌파 기반 실시간 피드백
> - 일일 뇌파 변화 추이 기록 및 누적 분석
>
> **▶ LINK BAND 미착용 시 (기본 모드)**
> - 상담/수업 예약 및 일정 관리
> - 음성 녹음 기반 자동 기록 및 AI 요약
> - 상담 기록지 관리 및 내담자 프로필
> - AI 요약 기반 세션 리포트 (뇌파 데이터 없이 텍스트 요약만 포함)
> - 셀프 트레이닝 콘텐츠 이용 (가이드 명상/호흡 훈련, 뇌파 피드백 없이 수행만)

**모니터링 대시보드 구성 요소:**

- 실시간 뇌파 신호 그래프 (Raw EEG waveform)
- AI 분석 지수: 집중도 / 이완도 / 스트레스 지수 / 뇌파 균형도
- 시계열 트렌드 차트 (세션 전체 변화 흐름)
- 명상 수업 시: 참여자 전체 상태 그리드 뷰 (한 화면에 다수 모니터링)
- 상태 변화 알림: 급격한 변화 감지 시 시각적 알림
- 세션 타이머 및 마커 기능 (주요 시점 태그)

#### 4.3.1 세션 유형별 모니터링 차이 (LINK BAND 착용 시)

| 구분 | 화면 구성 | 핵심 지표 |
|------|----------|----------|
| 1:1 상담 | 단일 내담자 상세 뷰 + 대화 타임라인 연동 | 스트레스 변화, 감정 전환점, 이완 구간 |
| 최면 상담 | 단일 내담자 상세 뷰 + 최면 단계 표시 | Alpha/Theta 비율, 의식수준, 이완도 변화 |
| 명상 수업 | 참여자 그리드 뷰 (N명 동시 모니터링) | 평균 이완도, 개인별 상태 컬러 코드 |

### 4.4 리포트 발행

세션 종료 후 AI가 자동으로 생성하는 개인화 리포트입니다. 상담사/명상 지도사용과 내담자/참여자용으로 구분됩니다.

#### 4.4.1 상담사/지도사용 리포트

- **세션 전체 요약:** AI가 생성한 세션 핵심 내용 요약
- **뇌파 분석 결과(선택):** 세션 전반 뇌파 변화 그래프 및 핵심 구간 분석 (LINK BAND 착용 시)
- **다음 세션 제안:** AI 기반 다음 세션 방향 추천
- **내담자 추이 분석:** 이전 세션 대비 변화도 분석 (LINK BAND 착용 시 뇌파 추이 포함)

#### 4.4.2 내담자/참여자용 리포트

- **세션 결과 요약:** 쉽게 이해할 수 있는 시각적 요약
- **나의 뇌 상태(선택):** 집중도, 이완도, 스트레스 지수 등 시각화 (LINK BAND 착용 시)
- **셀프 트레이닝 처방:** 상담사가 지정한 일일 트레이닝 과제
- **누적 리포트:** 세션별 변화 추이 그래프 (LINK BAND 착용 시 뇌파 추이 포함)

### 4.5 셀프 트레이닝

세션 간 기간에 내담자/참여자가 지속적으로 수행하는 자가 트레이닝 기능입니다.

> **셀프 트레이닝 흐름**
>
> ① 상담사가 트레이닝 프로그램 처방 (유형, 주기, 시간 설정)
> ② 내담자 웹에서 트레이닝 프로그램 확인 및 시작
> ③ LINK BAND 연결(선택) 후 가이드 명상/호흡 훈련 등 수행
> ④ 트레이닝 완료 기록 자동 저장 (LINK BAND 착용 시 뇌파 데이터 포함)
> ⑤ 다음 상담 시 상담사가 셀프 트레이닝 데이터 참고

#### 4.5.1 트레이닝 유형

- **가이드 명상:** 오디오 가이드에 따른 명상 수행 + 뇌파 측정(선택)
- **호흡 훈련:** 호흡 패턴 가이드 + 실시간 이완도 피드백(선택)
- **인지 훈련:** 집중력 향상 과제 + 뇌파 기반 난이도 조절(선택)
- **저널링:** 감정 기록 + 뇌파 상태와 감정 상관관계 분석(선택)

## 5. 사용자 여정 (User Flow)

### 5.1 상담사/명상 지도사 여정

전문가 사용자의 전체 서비스 이용 흐름입니다.

**PHASE 1: 세션 준비**

1. 로그인 → 대시보드 확인 (오늘의 예약, 알림, 통계 요약)
2. 새 세션 생성: 세션 유형 선택 → 일정 설정 → 내담자/참여자 초대
3. 이전 세션 기록 확인: 기록지 검토, 셀프 트레이닝 데이터 확인

**PHASE 2: 세션 진행**

4. 세션 시작: LINK BAND 연결 안내(선택) → 음성 녹음 동의 → 세션 시작
5. 실시간 모니터링: LINK BAND 연결 시 뇌파 대시보드 확인 / 미연결 시 세션 타이머만 표시
6. 주요 시점 마커: 특이 사항 발견 시 타임스탬프 태그 (메모 추가)

**PHASE 3: 세션 종료 및 후속**

7. 세션 종료: 녹음 종료 → AI 요약 자동 생성 (수 분 내)
8. 기록지 확인 및 수정: AI 생성 기록지 검토 → 보완 편집 → 저장
9. 리포트 발행: 내담자용 리포트 확인 → 발송 승인
10. 셀프 트레이닝 처방: 트레이닝 프로그램 선택 → 스케줄 설정 → 할당

### 5.2 내담자/참여자 여정

내담자/참여자의 전체 서비스 이용 흐름입니다.

**PHASE 1: 세션 참여**

1. 초대 알림 수신 → 웹 접속 → 회원가입/로그인
2. 예약 확인 및 승인 → 세션 대기
3. 세션 시작: LINK BAND 연결 안내(선택, 동의 시 BLE 웹 연결) → 세션 참여

**PHASE 2: 세션 후 확인**

4. 세션 종료 후 리포트 수신 알림
5. 개인 리포트 확인: 세션 요약, 뇌파 분석 결과(선택), 추이 그래프

**PHASE 3: 셀프 트레이닝**

6. 처방된 트레이닝 프로그램 확인
7. LINK BAND 연결(선택) → 가이드 명상/호흡 훈련 등 수행 → 결과 자동 기록
8. 일일 트레이닝 리포트 확인 → 누적 변화 추이 확인

## 6. 기술 아키텍처

### 6.1 시스템 구성도

웹 기반 SPA 플랫폼으로, Web Bluetooth API를 통해 LINK BAND와 직접 연동하며, 리얼타임 데이터는 WebSocket을 통해 처리합니다.

| 계층 | 기술 | 역할 |
|------|------|------|
| Frontend | React + TypeScript | SPA 구조, 대시보드 UI, 실시간 차트 |
| BLE 연동 | Web Bluetooth API | LINK BAND BLE GATT 연결 및 EEG 스트림 수신 |
| Realtime | WebSocket (Socket.IO) | 실시간 뇌파 데이터 전송 및 다자 모니터링 |
| Backend | Python (FastAPI) | API 서버, 세션 관리, 인증, 데이터 처리 |
| AI/ML | LLM API + EEG ML | STT 변환, 상담 요약, 뇌파 분석 (집중/이완/스트레스) |
| Database | PostgreSQL + TimescaleDB | 사용자/세션 데이터 + 시계열 뇌파 데이터 저장 |
| Storage | S3 Compatible | 음성 녹음 파일, 리포트 PDF, 미디어 파일 저장 |
| Infra | AWS / Docker | ECS/EKS 컨테이너 기반 배포, Auto Scaling |

### 6.2 기술 스택 상세

#### 6.2.1 Frontend

- React 18+ (TypeScript) — SPA 구조
- TanStack Query — 서버 상태 관리 및 캐싱
- Zustand — 클라이언트 상태 관리 (BLE 연결 상태, 실시간 데이터)
- Recharts / D3.js — 실시간 뇌파 차트 및 리포트 시각화
- Web Bluetooth API — LINK BAND BLE GATT 직접 연결
- Web Audio API — 브라우저 기반 음성 녹음

#### 6.2.2 Backend

- FastAPI (Python) — REST API + WebSocket 서버
- Celery + Redis — 비동기 작업 (STT 변환, AI 요약, 리포트 생성)
- Socket.IO — 실시간 뇌파 데이터 및 세션 상태 전송
- SQLAlchemy + Alembic — ORM 및 DB 마이그레이션

#### 6.2.3 AI/ML 파이프라인

- **STT:** Whisper API 또는 Clova Speech — 한국어 음성 변환
- **Speaker Diarization:** pyannote.audio — 화자 분리
- **상담 요약:** Claude/GPT API — 세션 요약 및 기록지 생성
- **EEG 분석:** 룩시드랩스 AI 분석 엔진 (SDK) — 집중도/이완도/스트레스 지수
- **리포트 생성:** LLM + 템플릿 엔진 — 개인화 리포트 PDF 생성

### 6.3 LINK BAND 웹 연동 아키텍처

Web Bluetooth API를 활용하여 브라우저에서 직접 LINK BAND와 BLE 통신합니다.

> **Web Bluetooth 연결 플로우**
>
> 1. 사용자 제스처로 BLE 스캔 시작 (`navigator.bluetooth.requestDevice`)
> 2. LINK BAND GATT Service UUID로 디바이스 필터링
> 3. EEG Characteristic 구독 (`startNotifications`)으로 실시간 데이터 수신
> 4. 수신된 Raw EEG 데이터를 WebSocket으로 백엔드 전송
> 5. 백엔드에서 AI 분석 후 결과를 WebSocket으로 프론트엔드에 반환

⚠️ **주의사항:** Web Bluetooth API는 Chrome/Edge 등 Chromium 기반 브라우저에서만 지원됩니다. Safari/Firefox는 미지원이므로 브라우저 호환성 안내가 필요합니다. 모바일 환경에서는 Android Chrome은 지원되지만, iOS Safari는 미지원이므로 추후 iOS 대응 전략이 필요합니다.

### 6.4 핵심 데이터 모델

시스템의 핵심 데이터 엔티티와 관계를 정의합니다.

| 엔티티 | 주요 필드 | 관계 |
|--------|----------|------|
| User | id, email, name, role, profile | 1:N Session, 1:N TrainingProgram |
| Session | id, type, status, scheduledAt, duration, hostId | N:1 User(host), N:M User(participant) |
| EEGRecord | id, sessionId, userId, timestamp, rawData, analysisResult — LINK BAND 착용 시에만 생성 | N:1 Session, N:1 User |
| SessionRecord | id, sessionId, transcript, aiSummary, markers[], counselorNotes | 1:1 Session |
| Report | id, sessionId, userId, type, content, pdfUrl | N:1 Session, N:1 User |
| TrainingProgram | id, clientId, counselorId, type, schedule, exercises[] | N:1 User(counselor), N:1 User(client) |
| TrainingLog | id, programId, completedAt, eegSummary(nullable, 선택), duration | N:1 TrainingProgram |

## 7. MVP 단계별 로드맵

개발 리소스를 고려하여 3단계 MVP 접근으로 점진적으로 확장해 나갑니다.

### MVP 1: 핵심 기반 구축 (8주)

> **목표:** 기본 세션 관리 + 실시간 모니터링 시스템 구축

1. 사용자 인증 시스템 (회원가입/로그인, 역할 구분)
2. 세션 CRUD (생성/수정/삭제, 세션 유형 3종 지원)
3. 내담자/참여자 관리 (프로필, 초대, 예약 확인)
4. Web Bluetooth LINK BAND 연결(선택) — 페어링, EEG 스트림 수신
5. 실시간 뇌파 모니터링 대시보드(선택) — 1:1 상담용 단일 뷰
6. 기본 EEG 분석 지표 시각화(선택) — 집중도/이완도/스트레스

### MVP 2: AI 기록 및 리포트 (6주)

> **목표:** 자동 기록 + AI 요약 + 리포트 발행 시스템

1. 음성 녹음 및 STT 변환 파이프라인
2. 화자 분리 (Speaker Diarization) 적용
3. AI 세션 요약 및 기록지 자동 생성 (LLM 기반)
4. 상담사용 세션 리포트 생성 및 조회
5. 내담자용 개인화 리포트 생성 및 발송
6. 명상 수업 다자 모니터링 그리드 뷰(선택)

### MVP 3: 셀프 트레이닝 및 고도화 (6주)

> **목표:** 셀프 트레이닝 + 데이터 연계 + 플랫폼 고도화

1. 셀프 트레이닝 프로그램 관리 (상담사 처방 기능)
2. 가이드 명상/호흡 훈련 콘텐츠 (오디오 + 실시간 피드백(선택))
3. 일일 트레이닝 기록 및 누적 변화 추이 분석
4. 상담 세션과 셀프 트레이닝 데이터 연계 뷰
5. 예약/알림 시스템 (Email/Web Push)
6. 대시보드 고도화 (통계, 추이 분석, 내담자 타임라인)

### 7.1 전체 타임라인 요약

| 단계 | 핵심 목표 | 기간 | 주요 산출물 |
|------|----------|------|-----------|
| **MVP 1** | 핵심 기반 구축 | 8주 | 세션 관리 + 실시간 모니터링 |
