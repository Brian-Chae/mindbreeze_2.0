## 1. 프로젝트 개요

### 1.1 서비스 소개
**MIND BREEZE**는 뇌파(EEG) 기반 심리상담 · 명상 통합 서비스 플랫폼입니다.

- **핵심 가치**: AI 기반 자동 기록/요약, 과학적 멘탈 케어, LINK BAND 2.0 뇌파 측정(선택)
- **서비스 모델**: B2B (상담사/명상 지도사가 고객, 내담자는 최종 사용자)
- **플랫폼**: 웹(관리자/내담자) + 모바일(내담자)
- **서비스 플로우**: 진단(설문) → 케어(과제 수행) → 리포트(상담 결과) — AI가 전 과정 지원

### 1.2 디자인 리디자인 대상
로그인 및 회원가입(온보딩) 플로우의 전체 화면 디자인 개선

  - 로고 "MIND BREEZE" + psychology 아이콘
  - 슬로건: "마음의 평화를 과학으로 만나다"
  - 서비스 설명 텍스트
  - 특장점 3개 (AI 기반 기록, LINK BAND, 실시간 모니터링)

### 2.3 현재 디자인의 개선 포인트

- 브랜드 패널의 메시 그라데이션이 너무 어둡고 무거운 느낌
- 폼 영역이 단조롭고 일반적인 웹 폼 수준
- 브랜드 아이덴티티가 "심리상담 + AI + 과학" 느낌을 충분히 전달하지 못함
- 온보딩 스텝 간 시각적 연결성이 부족
- 모바일 반응형 고려 필요

## 4. 디자인 개선 방향

### 4.1 핵심 컨셉 키워드

- **과학적 신뢰감**: 뇌파/AI 기반 서비스임을 느끼게 하는 클린한 테크 느낌
- **따뜻한 케어**: 심리상담 서비스인만큼 차갑지 않고 따뜻하고 안정감 있는 톤
- **프리미엄 모던**: SaaS급 세련된 UI, 넉넉한 여백, 절제된 장식
- **AI 네이티브**: AI가 핵심인 서비스임을 시각적으로 전달 (subtle AI elements)
- **접근성**: 상담사(40-50대 포함)와 내담자(20-50대) 모두 편안하게 사용

### 4.2 원하는 개선 포인트

1. **브랜드 패널**: 현재 어두운 메시 그라데이션 → 더 밝고 세련된 비주얼로 변경. 일러스트레이션이나 3D 요소, 또는 미니멀한 패턴 활용 고려
2. **로그인 폼**: 더 넉넉한 여백, 모던한 입력 필드 스타일, 마이크로 인터랙션 힌트
3. **소셜 로그인**: 더 깔끔하고 직관적인 배치
4. **온보딩 플로우**: 스텝별 진행 표시를 시각적으로 더 명확하게, 각 스텝의 일관된 레이아웃
5. **반응형**: 모바일에서도 자연스럽게 보이는 구조

### 4.3 참고할 디자인 스타일

- Calm.com — 따뜻한 웰니스 느낌
- Linear.app — 모던 SaaS 로그인
- Notion — 클린하고 미니멀한 온보딩
- Headspace — 친근하면서도 과학적인 톤

## 5. 화면별 요구사항

### 5.1 로그인 화면

**레이아웃**: 1440x900 (데스크톱), 좌/우 2분할 또는 중앙 정렬

**필수 요소:**
- 로고 (MIND BREEZE + psychology 아이콘)
- 브랜드 슬로건 또는 비주얼
- 이메일 입력 필드
- 비밀번호 입력 필드
- 비밀번호 찾기 링크
- 로그인 CTA 버튼
- "또는" 구분선
- Google 소셜 로그인 버튼
- Kakao 소셜 로그인 버튼
- 회원가입 링크
- 저작권 푸터

**디자인 요구:**
- 브랜드 컬러(#5F0080 보라) 기반이되 너무 무겁지 않게
- AI/과학적 느낌을 주는 시각 요소 (뇌파 웨이브, 뉴로 패턴, 또는 추상적 그래픽)
- 입력 필드는 넉넉한 패딩, 부드러운 포커스 상태
- CTA 버튼은 시각적 무게감 있게

### 5.2 역할 선택 화면

**레이아웃**: 중앙 정렬, 2개 카드 선택

**필수 요소:**
- "어떤 역할로 가입하시겠어요?" 타이틀
- 상담사/지도사 카드 (아이콘 + 타이틀 + 설명)
- 내담자/참여자 카드 (아이콘 + 타이틀 + 설명)
- 뒤로가기/로그인 링크

### 5.3 상담사 회원가입 (4단계)

**공통 레이아웃:** 좌측 브랜드/진행률 패널 + 우측 폼

**Step 1 — 기본 정보:**
- 이름, 이메일, 비밀번호, 비밀번호 확인
- 진행률: 1/4

**Step 2 — 상세 정보:**
- 성별, 생년월일, 소속 기관 주소
- 경력 사항 (자유 입력, 추가 가능)
- 전문 분야 (자유 입력, 태그 형태)
- 진행률: 2/4

**Step 3 — 전문 자격 인증:**
- 자격증 종류 선택
- 자격증 번호 입력
- 파일 업로드 영역
- 진행률: 3/4

**Step 4 — 프로필 완성:**
- 프로필 사진 업로드
- 한줄 소개 텍스트
- 진행률: 4/4

**프로필 확인 화면:**
- 완성된 프로필 카드 미리보기
- 상담사 초대 코드 (6자리) 표시
- "시작하기" CTA

### 5.4 내담자 회원가입 (4단계)

**Step 1 — 기본 정보:** 이름, 이메일, 비밀번호
**Step 2 — 상세 정보:** 성별, 생년월일, 불편 영역(복수 선택), 서비스 유형 선택
**Step 3 — 프로필 만들기:** 사진, 닉네임
**Step 4 — 상담사 지정:** 6자리 초대 코드 입력 (OTP 스타일), 상담사 정보 자동 표시, 연결 확인
