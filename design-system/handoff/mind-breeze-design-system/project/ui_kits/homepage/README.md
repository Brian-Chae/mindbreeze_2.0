# MIND BREEZE — Homepage UI Kit

A click-through recreation of the **MIND BREEZE** marketing homepage. Based on the existing site at https://mindbreeze.kr and the Figma `/Homepage` frames.

Design width: **1440** (the site scales fluidly; cards use a 3-up grid that becomes 1-up on mobile). Cream background, white feature cards, lavender capsule eyebrows. Korean-first copy lifted directly from Figma.

## Files

- `index.html` — full assembled homepage with sticky nav, hero, three feature cards, process section, CTA, footer
- `Nav.jsx` — sticky top navigation with brand mark + menu + CTA
- `Hero.jsx` — cream hero with bilingual headline + eyebrow + lead CTA + brain-wave symbol
- `FeatureCards.jsx` — three "evidence" cards (특허 / CES / AI 데이터)
- `ProcessSection.jsx` — four-step process (등록 → 측정 → 분석 → 리포트)
- `CTASection.jsx` — purple slab with white symbol motif + signup CTA
- `Footer.jsx` — dark footer with brand info + links
