## 0. 의사결정 한 줄 (North Star)

> **"과학으로 검증된 평온함 — 차분한 자연의 톤으로 신뢰를 만드는 멘탈 케어 플랫폼."**

상담사가 안심하고 내담자를 모실 수 있는 공간이며, 내담자가 자기 자신을 마주할 수 있는 따뜻한 인터페이스. 화려함이 아닌 **고요한 권위(quiet authority)** 가 본 브랜드의 핵심 자산이다.

## 1. 브랜드 포지셔닝

### 1.1 카테고리

뇌파(EEG) 기반 심리상담·명상 통합 B2B SaaS 플랫폼.

### 1.2 차별화 한 줄

> **"AI가 기록하고, 뇌파가 증명하며, 사람이 결정한다."**

### 1.3 약속 (Promise)

| 대상 | 약속 |
|------|------|
| **상담사** | 기록의 부담을 덜고, 통찰의 시간을 돌려준다. |
| **내담자** | 당신의 마음에 일어나는 일을 함께 본다. |
| **기관(센터)** | 자격·신뢰·운영을 자동으로 보장한다. |

### 1.4 비포지셔닝 (We are NOT)

- ❌ 의료기기·진단 도구가 아니다 (LINK BAND는 의료기기 아님)
- ❌ "마음을 고친다" 같은 결과 약속을 하지 않는다
- ❌ 게이미피케이션·뱃지·연속 출석 같은 도파민 UX를 쓰지 않는다
- ❌ 어두운 보라색 사이버 펑크 미학 아님 (variant 시안과 결별)
- ❌ 영문 가독성을 한글보다 우선하지 않는다 (한국 사용자 1순위)

## 2. 브랜드 페르소나

브랜드를 사람으로 의인화한다면:

> **"40대 후반의 식물학자 출신 임상심리학자. 항상 차분하고 정확하며, 모르는 것은 모른다고 말하고, 안다는 것은 근거와 함께 말한다. 책상 위에는 작은 화분이 있고, 메모는 만년필로 한다. 환자가 우는 순간에는 말 대신 침묵으로 옆에 있어준다."**

## 3. Voice & Tone

### 3.1 항상 (Always)

- **차분하다** — 느낌표를 거의 쓰지 않는다
- **정확하다** — 수치·근거·출처를 동반한다
- **명확하다** — 의학 전문 용어 없이 설명한다
- **존중한다** — "환자"가 아니라 "내담자", 명령형이 아니라 권유형

### 3.2 절대 (Never)

- **약속하지 않는다** — "스트레스가 사라집니다" ❌ → "스트레스 지표를 관찰합니다" ✅
- **유행어를 쓰지 않는다** — "혁신적", "최첨단", "게임체인저" ❌
- **이모지를 본문에 쓰지 않는다** — 시스템 메시지·이벤트 라벨에만 제한 사용
- **느낌표 두 개 이상** — 한 문장에 `!` 하나도 드물게

### 3.3 마이크로카피 예시

| 상황 | ❌ 일반적 | ✅ MIND BREEZE |
|------|----------|---------------|
| 세션 시작 | "지금 바로 시작하세요!" | "준비되시면 시작합니다." |
| 녹음 권한 | "녹음을 허용해주세요!" | "기록을 위해 마이크 사용 동의가 필요합니다." |
| EEG 미연결 | "LINK BAND를 연결하지 못했습니다 😢" | "지금은 LINK BAND 없이 진행합니다." |
| 분석 완료 | "분석이 완료되었어요!" | "리포트가 준비되었습니다." |
| 자격 인증 통과 | "축하합니다! 인증 완료!" | "자격 확인이 완료되었습니다." |
| 빈 화면 | "아직 아무것도 없어요" | "첫 세션을 등록해 시작해보세요." |
| 에러 | "오류가 발생했어요!" | "잠시 후 다시 시도해 주세요. 문제가 계속되면 도움말을 참고하세요." |

### 3.4 한국어 · 영어 혼용 규칙

- **기본은 한국어.** 영문 약어는 정착된 것만 (`LINK BAND`, `EEG`, `STT`, `AI`)
- 신조어·마케팅 표현은 한국어로 번역해 사용 (예: "온보딩" ✅ / "그로스 해킹" ❌)
- 한자 병기는 의학 용어 첫 등장 시 1회만 (`내담자(來談者)`) — 이후 한글

## 4. 비주얼 시그니처 — Breath Circle

브랜드의 **단 하나의 잊히지 않는 모티프**.

```
        ·  ·  ·  ·  ·
      ·               ·
    ·     ◯           ·
    ·       ○         ·
    ·         ●       ·
      ·               ·
        ·  ·  ·  ·  ·
```

**Breath Circle**은 세 겹의 동심원이 호흡 리듬(4초 들숨 · 4초 날숨)에 맞춰 부드럽게 확장·수축한다. 다음 위치에 일관되게 등장한다:

| 위치 | 기능 |
|------|------|
| 앱 로더 | 로딩 인디케이터 (스피너 대체) |
| 빈 상태(Empty state) | "잠시의 여백" 상징 |
| 세션 시작 화면 | 시각적 호흡 가이드 |
| 인증 등급 뱃지 (Verified+) | 검증된 신뢰의 표식 |
| 다크 모드 진입 트랜지션 | 부드러운 상태 변화 |

**기술적 구현**: SVG + CSS `@keyframes breathe`. WebGL·캔버스 없이 100% CSS로 구현 가능해야 한다 (저사양 디바이스 보장).

`prefers-reduced-motion: reduce` 환경에서는 정지된 동심원으로 표시한다.

## 5. 컬러 철학

### 5.1 원칙

1. **자연에서 빌려온다** — 식물·물·새벽 하늘. 합성·네온 금지.
2. **저채도 + 1포인트** — 화면 대부분은 차분한 베이지·세이지. 강조는 dawn coral 또는 morning sky 단일 색.
3. **빨강을 피한다** — 위험·경고는 `clay`(따뜻한 흙)와 `earth`(차분한 적갈색). 응급 알람용 순수 red는 의도적으로 제외.
4. **명도 대비 우선** — WCAG AA 4.5:1 필수. 채도가 아닌 명도로 위계를 만든다.

### 5.2 팔레트 (Light 기준)

```

 │    oat     │     │ deep slate │     │   sage     │     │ dawn coral │
 ├────────────┤     ├────────────┤     ├────────────┤     ├────────────┤
 │   #FAF8F2  │     │   #5C6B7A  │     │   #4A6B5A  │     │   #A5C5D6  │
```

### 5.3 다크 모드 원칙

- 베이스를 **순수 검정이 아닌 "숲의 어둠"** (`#1A1D1A`)으로
- 사주 sage는 명도를 올려 가독성 확보 (`#A8C0A7`)
- 액센트는 light/dark 동일 (브랜드 일관성)
- 배경 위계는 명도 5%씩 단계적으로

자세한 명세는 [`design-system/tokens/system/`](../tokens/system/) 참조.

## 6. 타이포그래피 철학

### 6.1 선택과 이유

| 역할 | 폰트 | 이유 |
|------|------|------|
| **Display (영문)** | **Fraunces** | 따뜻한 옵티컬 사이즈 세리프. 식물학 저널·자연 다큐멘터리 분위기. "현대적이되 영속적". 무료(Google Fonts). |
| **Body (영문)** | **Inter** | 한국에서 가장 검증된 산세리프. Pretendard와 메트릭이 일치해 한영 혼용 시 베이스라인 어긋남 없음. |
| **Body (한글)** | **Pretendard Variable** | 한국어 디지털 환경 사실상 표준. Inter와 시각적으로 짝꿍. |
| **Mono (기술 라벨)** | **JetBrains Mono** | 기술적 정확성이 필요한 곳에만 최소 사용 (디바이스 ID, EEG 채널명, 시간 코드 등). |

**왜 Söhne/Recamier/GT America 아닌가**: 모두 유료. 우리는 무료 + 한국어 환경에 최적화된 조합을 우선한다. Brand가 폰트의 화려함이 아닌 **여백과 톤**에서 나오기를 의도한다.

### 6.2 사용 원칙

- **Display는 24px 이상에서만** — 작은 크기에서 세리프는 가독성을 해친다
- **Mono는 본문에 쓰지 않는다** — 오직 메타 라벨, 코드, 시간 스탬프, 디바이스 식별자
- **굵기는 4단계만** — 300(Light, 큰 헤드라인 한정) / 400(Regular) / 500(Medium) / 700(Bold)
- **이탤릭 거의 안 쓴다** — 인용 시에만

자세한 스케일은 [`design-system/tokens/system/typography.json`](../tokens/system/) 참조.

## 7. 형태(Shape) · 모서리

```
sm  ━━ 8px   ━━ 작은 칩, 태그
md  ━━ 12px  ━━ 입력 필드, 작은 카드
lg  ━━ 20px  ━━ 카드, 모달 (Clinical Garden의 시그니처 곡률)
xl  ━━ 32px  ━━ 히어로 카드, 큰 컨테이너
pill ━ 999px ━━ 버튼, 뱃지, 채팅 버블 ─ "유기적 곡선"
```

**원칙**: 직각보다 곡률. 모든 표면은 손이 닿을 수 있게 부드럽다. 단, 표·테이블·기술 라벨처럼 정확성이 필요한 곳은 `sm`/`md`로 절제.

## 8. 공간(Spacing) · 그리드

- **기본 단위**: 4px
- **스케일**: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 96, 128
- **그리드**: 12-column, gutter 24px (데스크탑) · 16px (모바일)
- **컨테이너 최대 폭**: 1280px (앱), 1440px (마케팅)
- **여백 철학**: **여백은 콘텐츠보다 중요하다**. 카드 내부 padding은 절대 줄이지 않는다.

## 9. 모션(Motion) 원칙

### 9.1 The Three Speeds

| 종류 | duration | easing | 사용처 |
|------|---------|--------|--------|
| **Instant** | 100-150ms | `ease-out` | 호버, 포커스, 토글 |
| **Smooth** | 200-300ms | `cubic-bezier(0.16, 1, 0.3, 1)` | 페이지 전이, 모달, 카드 펼침 |
| **Breath** | 4000ms | `ease-in-out` infinite | Breath Circle, 로딩, 명상 가이드 |

### 9.2 금지된 모션

- ❌ 튕김(bounce) · 출렁임(elastic) — 진지함을 해친다
- ❌ 패럴랙스 과다 사용 — 멀미·산만함
- ❌ 자동 재생 비디오 — 사용자 통제권 침해
- ❌ 모든 호버에 반응 — 의미 있는 곳에만

### 9.3 접근성

`prefers-reduced-motion: reduce` 매체 쿼리를 **모든 애니메이션에서 존중**한다. Breath Circle도 정지된 동심원으로 대체.

## 10. 이미지 · 일러스트레이션

### 10.1 사진

- **있는 그대로의 자연광** — 스튜디오 라이팅·플래시 금지
- **사람의 얼굴은 부분만** — 손, 옆모습, 실루엣
- **식물·물·창** — 자주 등장하는 환경 요소
- **금지**: 활짝 웃는 스톡 사진, 정장의 비즈니스맨, 회의실 풍경, 뇌의 X-ray 이미지

### 10.2 일러스트레이션

- **선 두께 단일** — 1.5px stroke, no fill (또는 매우 옅은 fill)
- **식물 라인 드로잉** — 잎, 줄기, 새순 모티프
- **추상 곡선** — EEG 파형은 "잎맥"처럼, "리본"처럼

### 10.3 아이콘

- **Phosphor Icons** (Regular weight 기본, Duotone은 강조에만)
- 사이즈: 16, 20, 24, 32px
- 자체 제작 아이콘: Breath Circle, "검증 잎(verified leaf)" 뱃지, EEG-as-leaves 모티프

## 12. 접근성 (Accessibility)

브랜드의 **윤리적 요구사항**. 미관보다 우선한다.

| 항목 | 기준 |
|------|------|
| 본문 대비 | 4.5:1 (AA) 최소, 7:1 (AAA) 권장 |
| 큰 텍스트(18.66px+) | 3:1 최소 |
| 비텍스트 UI | 3:1 최소 |
| 포커스 링 | 시각적으로 명확 (2px solid `brand/deep`, offset 2px) |
| 키보드 네비게이션 | 모든 인터랙티브 요소 도달 가능 |
| 스크린 리더 | aria 라벨 필수, semantic HTML 우선 |
| 폰트 크기 | 본문 16px 미만 금지 (Caption만 14px 허용) |
| 라인 높이 | 본문 1.6, 헤드라인 1.2 |

## 13. 차별화 요소 정리 — 잊히지 않는 것들

3. **Breath Circle 모티프** — 모든 로더·뱃지·전환에 반복 등장하는 시그니처.
4. **Fraunces Display** — 세리프 헤드라인의 "출판물 같은 권위".
5. **유기적 모서리 (radius 20px+)** — 직각 없는 부드러운 표면.
6. **여백 우선** — 정보 밀도가 아닌 호흡감.
7. **이모지·느낌표 절제** — 한 화면에 이모지 0개가 기본.
8. **자격 인증 뱃지의 잎 모티프** — `Verified+` 뱃지에 작은 새순.
